##启动方式
- npm start：node server提供首页入口及模拟api server，webpack dev server提供静态文件
- npm run devserver: 启动webpack - HotDevServer
- npm run biuld: 生成生产文件
- npm run test: 运行全部单元测试
- npm run atest [case-name]: 运行特定的一个或几个单元测试
